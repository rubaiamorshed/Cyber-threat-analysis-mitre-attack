# Cyber-threat-analysis-mitre-attack
Cybersecurity project demonstrating threat analysis and technique mapping using the MITRE ATT&amp;CK framework


## 📌 Project Summary

This project demonstrates hands-on experience with the MITRE ATT&CK framework by leveraging the ATT&CK Navigator to analyze, map, and compare adversary behaviors.

The project focuses on identifying and visualizing techniques used by advanced threat groups, enabling structured threat intelligence analysis and security insights.

---

## 🎯 Key Highlights

* Applied **MITRE ATT&CK framework** to real-world threat actor analysis
* Built and analyzed **custom ATT&CK Navigator layers**
* Performed **comparative analysis of adversary techniques**
* Demonstrated **threat modeling and detection thinking**
* Produced **professional cybersecurity documentation** suitable for industry use

---

## 🧪 Scope of Analysis

Two threat groups were selected for behavioral comparison:

* APT18 (Cyber espionage-focused)
* APT38 (Financially motivated attacks)

The analysis highlights:

* Shared attack techniques (overlapping behaviors)
* Unique techniques (group-specific tactics)
* Visualization of attack patterns using scoring and color mapping

---

## 🛠️ Technical Implementation

Using the ATT&CK Navigator, the following tasks were completed:

* ✅ Created new layers for individual threat groups
* ✅ Assigned technique scores and visual indicators
* ✅ Exported and imported JSON layer files
* ✅ Combined multiple layers for comparative analysis
* ✅ Validated techniques against official ATT&CK data
